  is the systemic fix and is worth raising in priority.

  Without native sub-issue links the Goal progress bars under-report.
- **Status drift** (#590, and the related #586) — doc headers lag code reality across ADR-025/026/027.
  The P0 doc fix addresses ADR-025; the others are worth a sweep.
- **Silent security gap** — Rust M5 (ADR-025 Phase 2) ships in CI with no RBAC interceptor. Harmless while
  Go M5 is production, but any cutover/shadow to Rust M5 would expose an unauthenticated mutation surface.
- **Merged-with-red-history watch** — #661 merged after a prior red rust check; the post-merge `main` run
  settled with rust + go green — **but the docker image-publish jobs failed** (below).
- **✅ `main` docker image-publish — Go toolchain skew (fixed by #669)** — run `28560416147` (the #641 merge):
  `docker (metrics)` and `docker (management)` failed with
  `go: module ../gen/go requires go >= 1.26 (running go 1.25.11; GOTOOLCHAIN=local)`. The workspace
  (`go.work` → `go 1.26.1`) and the `go` CI job (setup-go 1.26) had moved to 1.26; the service Dockerfiles
  still pinned `golang:1.25-bookworm` (orchestration `1.22`). The ADR-031 merge regenerated `gen/go`
